# Checklist — Smoke Test E-commerce

> Execução rápida após cada deploy para validar funcionalidades críticas.

**Versão:** 1.0 | **Tempo estimado:** 15 min

---

## Autenticação
- [ ] Login com credenciais válidas funciona
- [ ] Logout encerra a sessão corretamente
- [ ] Redirecionamento após login está correto

## Catálogo de produtos
- [ ] Página inicial carrega sem erros
- [ ] Busca de produto retorna resultados
- [ ] Página de detalhe do produto abre corretamente
- [ ] Imagens dos produtos carregam

## Carrinho
- [ ] Adicionar produto ao carrinho
- [ ] Alterar quantidade no carrinho
- [ ] Remover produto do carrinho
- [ ] Total do carrinho é calculado corretamente

## Checkout
- [ ] Fluxo de checkout inicia corretamente
- [ ] Campos de endereço são validados
- [ ] Seleção de frete funciona

## Performance básica
- [ ] Página inicial carrega em menos de 3 segundos
- [ ] Nenhum erro 500 no console do navegador
